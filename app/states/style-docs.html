Alerts

The alerts directive creates an alert. It generates the same markup as bootstrap alerts.

Options:

  • type (required): success, info, warning, danger
  • dismissible (optional | defaults to false): true, false

HTML source (AngularJS directive)

<alert type="success">Success</alert>
<alert type="info">Info</alert>
<alert type="warning">Warning</alert>
<alert type="danger">Danger</alert>
<alert type="info" dismissible="true">Dismissible</alert>

Output

Success Info Warning Danger Dismissible
